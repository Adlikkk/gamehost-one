# Privacy Policy

GameHost ONE does not collect personal data.

## Summary

- Telemetry is opt-in and disabled by default.
- No IP addresses are stored.
- No server content is tracked.
- No third-party analytics are used.
- All data stays on your device unless you explicitly export it.

## What We Collect

If you enable anonymous usage analytics, the app records the following events locally:

- server_created
- crash_occurred

These events contain no personal data and do not include server content.

## Data Storage

By default, analytics and crash reports are stored locally in your AppData folder.
No data is uploaded automatically.

## Updates

Update checks use the GitHub Releases API. No personal data is sent.

## Contact

For questions, open an issue in the repository.

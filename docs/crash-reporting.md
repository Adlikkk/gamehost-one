# Crash Reporting

Crash reporting in GameHost ONE is opt-in and disabled by default.

## What Is Captured

- Stack trace
- App version
- OS
- Timestamp

## Storage

Crash reports are saved locally at:

AppData/GameHostOne/crashes/

## Export

On the next launch after a crash, GameHost ONE can offer to export a crash report as a ZIP file.
No crash data is uploaded automatically.

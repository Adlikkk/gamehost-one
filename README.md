# GameHost ONE

GameHost ONE is a desktop application for managing local game servers, focused on Minecraft. It runs fully on your machine and keeps your data local.

## Features

- Java auto-management
- One-click server setup
- Modpack sync
- Resource monitoring
- Tray integration

## Tech Stack

- Tauri
- Rust
- React
- TailwindCSS

## Privacy

- No telemetry by default
- No tracking
- No ads
